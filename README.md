# project_algeo2025
to setup:
```bash
python src/preprocess.py
```
```bash
python src/train.py
```
to use:
```bash
python src/main {audioFile path}
```
ex.
```bash
python src/main all_recordings/Omen_Stone.wav
```

note:
make sure you intall some python libraries needed, you can use
```bash
pip install -r requirements.txt
```
